# Recipe Management API

This is a Django REST Framework API for managing recipes.

## Features
- User authentication (Token Authentication)
- CRUD operations for recipes
- Categorize recipes
- Add ingredients to recipes

## Tech Stack
- Python
- Django
- Django REST Framework
- SQLite (development)
